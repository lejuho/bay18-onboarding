# BAY 18기 온보딩 2교시 — 인터랙티브 강의 사이트

매주 30분 온보딩 수업용 인터랙티브 HTML 슬라이드 모음. 정적 사이트라 서버 없이 Vercel에 그대로 배포됩니다.

## 구조

```
index.html      허브 — 주차 목록, 본인 진행 현황, 이름 저장
w1.html         1주차 · 블록체인과 상태 머신 (7/23)
w2.html         2주차 · 트랜잭션 검증과 전파 (7/30)
w3~w6.html      (추가 예정)
assemble.html   6주 산출물 → 트랜잭션 생애주기 지도 조립
config.js       구글폼 제출 설정 (아래 참고)
```

- 각 주차의 3분 산출물은 **localStorage에 자동 저장**되며, 같은 도메인의 `assemble.html`이 자동으로 읽어 지도를 만듭니다.
- 다른 기기에서 작성한 산출물은 조립 페이지의 붙여넣기 기능으로 합칠 수 있습니다 (`[BAY18-W#]` 태그 기준 자동 분류).

## Vercel 배포

1. 이 폴더를 GitHub 레포로 push
2. [vercel.com](https://vercel.com) → Add New Project → 해당 레포 Import
3. Framework Preset: **Other**, 빌드 설정 없음 → Deploy

이후 파일 수정 후 `git push`만 하면 자동 재배포됩니다.

## 구글폼 제출 연동 (config.js)

"구글폼으로 제출" 버튼은 산출물이 미리 채워진 폼을 새 탭으로 엽니다. 설정 전에는 자동으로 "복사" 동작으로 대체되므로, 설정 없이 배포해도 동작합니다.

### 1) 폼 만들기 (질문 5개)

| 질문 | 유형 | 비고 |
|---|---|---|
| 주차 | 단답형 | W1~W6, 자동 입력됨 |
| 활동기수 | 단답형 | 운영 단위. config.js의 term 값이 자동 입력됨 |
| 가입기수 | 단답형 | 참여자 소속(17기/18기). 허브에서 한 번 선택하면 자동 입력됨 |
| 이름 | 단답형 | 허브에서 한 번 입력하면 자동 입력됨 |
| 산출물 | 장문형 | 자동 입력됨 |

활동기수는 "이번 학기 18기 온보딩" 같은 운영 단위이고, 가입기수는 참여자 개인의 소속입니다(17기 신입이 이번 학기까지 활동하는 경우처럼 혼재 가능). 제출 시각은 폼이 자동 기록하므로 별도 항목이 필요 없습니다. 드롭다운은 prefill 값이 옵션 텍스트와 **정확히 일치**해야 채워지므로 전부 단답형을 권장합니다.

### 2) entry ID 확인

폼 편집 화면 우측 상단 ⋮ → **미리 채워진 링크 받기** → 각 칸에 아무 값이나 입력 → 링크 복사.
복사된 URL에서 `entry.1111111111=...` 형태의 ID 3개를 확인합니다.

### 3) config.js에 붙여넣기

```js
window.BAY18_CONFIG = {
  formAction: "https://docs.google.com/forms/d/e/1FAIpQLxxxx/viewform",
  term: "18기",
  fields: {
    week:   "entry.1111111111",
    term:   "entry.2222222222",
    member: "entry.3333333333",
    name:   "entry.4444444444",
    output: "entry.5555555555"
  }
};
```

학생이 직접 하는 건 허브(index.html)에서 이름 입력 + 가입 기수 선택 한 번뿐이고, 이후 매주 제출은 전부 자동으로 채워집니다. 응답은 스프레드시트에 `시각 / 주차 / 활동기수 / 가입기수 / 이름 / 산출물`로 쌓이므로 가입기수 필터로 17기·18기를 구분해 볼 수 있고, 다음 학기에는 config.js의 term만 바꾸면 같은 폼을 재사용할 수 있습니다.

## 산출물 텍스트 규격

```
[BAY18-W1] 상태 머신
S0 현재 상태: …
TX 트랜잭션: …
S1 새 상태: …
순서 문제: …
```

새 주차를 추가할 때는 이 규격(`[BAY18-W#] 제목` + `키: 값` 줄)을 유지하면 조립 페이지가 자동으로 인식합니다.
